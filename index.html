tudo sobre css

Cascading Style Sheets é uma linguagem de estilo utilizada para definir a apresentação de elementos HTML em uma página web.

CSS não é uma linguagem de programação.É uma linguagem de estilo.

o css consegue definir:

  • cores
  • tamanho
  • posicionamento
  • e muito mais...

formas de trabalhar com css

existem 3 formas de trabalhar com css:

  1. css inline
  2. css interno
  3. css externo

css inline

o css inline é quando você aplica o estilo diretamente no elemento HTML usando o atributo "style".

css interno

o css interno é quando você define os estilos dentro de uma tag no cabeçalho do documento HTML.

css externo

o css externo é quando você cria um arquivo separado com a extensão .css e o vincula ao seu documento HTML usando a tag
